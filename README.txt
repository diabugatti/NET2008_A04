1. Description:
  This application is about two computers playing a game of Rock Scissors Paper and telling the result.
 You can run the app using Play with Docker at "https://labs.play-with-docker.com/", using the pull command.
 After signing in, you will add new instance, type "docker pull minhwan/net2008_a04:latest" in the terminal and
 type "docker run minhwan/net2008_a04". Then the app will run.
